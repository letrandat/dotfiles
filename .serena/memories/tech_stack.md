# Tech Stack

## Primary Languages

### Lua
- **Usage**: Neovim configuration
- **Framework**: LazyVim (Lazy.nvim plugin manager)
- **Version**: Modern Lua (Neovim embedded)

### Shell Scripts
- **Bash**: Setup scripts, utilities
- **Zsh**: Interactive shell configuration
- **Framework**: Oh My Zsh

## Tools & Applications

### Development Environment

- **Neovim**: Primary text editor with LazyVim configuration
- **VS Code**: GUI editor with Vim keybindings
- **Windsurf**: Alternative VS Code-based editor
- **Antigravity**: Another editor in the ecosystem

### Terminal Ecosystem

- **tmux**: Terminal multiplexer with custom sessionizer
- **Ghostty**: Terminal emulator
- **Kitty**: Alternative terminal emulator
- **Starship**: Shell prompt customizer

### Version Control

- **Git**: Version control system
- **LazyGit**: Terminal UI for Git
- **Git worktrees**: Multi-branch workflow support

### Package Management

- **GNU Stow**: Symlink farm manager for dotfiles
- **Homebrew**: macOS package manager
- **Oh My Zsh**: Zsh framework and plugin manager
- **Lazy.nvim**: Neovim plugin manager

### AI Coding Assistants

- **Codeium**: AI code completion
- **Amp**: AI coding assistant

### Other Tools

- **IntelliJ IDEA**: IDE with IdeaVim plugin
- **kubectl/helm**: Kubernetes tools (zsh plugins)
- **openskills**: Agent skills manager

## Platform

- **Primary OS**: macOS (Darwin)
- **XDG Base Directory**: Enforced via `$XDG_CONFIG_HOME`
- **Architecture**: Cross-platform with macOS-specific Application Support handling
